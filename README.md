
# 🚀 API Time Machine

> Record, replay, and analyze API traffic with real-time visualization.

## ✨ Features
- Proxy-based API logging
- Real-time dashboard (WebSocket)
- Latency analytics (Chart.js)
- Replay & debugging support
- Lightweight & fast

## 🚀 Quick Start

```bash
git clone https://github.com/YOUR_USERNAME/api-time-machine.git
cd api-time-machine
npm install
npm start
```

Open:
http://localhost:3000/dashboard

## 🧪 Example
```bash
curl http://localhost:3000/posts
```

## 🧠 Use Cases
- Debug APIs
- Monitor performance
- Analyze latency trends

## 🛠 Tech Stack
Node.js, Express, SQLite, WebSocket, Chart.js
